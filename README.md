2. download the independencies by running **npm i**

3. Run nodemon to see your live server via local host in your browser.

## Setup

### Installed dependencies

    npm i dotenv
    npm i express
    npm i jest 
    npm i supertest
    npm i mongoose 
    npm i cors
    npm i base-64
    npm i bcrypt
    npm i morgan
    npm i @code-fellows/supergoose
    npm i jsonwebtoken

### .env requirments

    * PORT - port number
    * MONGO_URI
    * SECRET

### Running the application

* npm start || nodemon

* Endpoints:
  * /
  * /signup
  * /signin
  * /users
  * /sercet
  * /api/v1/clothes
  * /api/v1/food
  * /api/v2/clothes
  * /api/v2/food

### Tests

    * Unit Tests: npm run test
    * Lint Tests: npm run lint
